A. General Information

   Cut-off date:  Cut-off date for this release of WIHS Public Data Set
        is September 30, 2015.  Data covered up to visit 42 of WIHS.

   Data Media: Data is available for download as ASCII or CSV files

B. Associated Questionnaire Forms and Documentation

   Forms:

   Forms are in Microsoft Word documents or PDF files.

   Other documentation:

   WIHS directory of investigators and archives of publication
   can be found in WIHS web site (www.wihshealth.org).

\sasinp directory
 --SAS data input statement for each data file (<FileName>.dat) with name of <FileName>.inp)

Sample SAS syntax for reading in a <FileName>.dat file:
DATA f22hx;
   INFILE "d:\WIHS_P16\data\f22hx.dat" TRUNCOVER LRECL = 5000;
   %INCLUDE "d:\WIHS_P16\sasinp\f22hx.inp";
RUN;
Please refer to support.sas.com for SAS assistance.
